# Команди, які використовувалися на занятті

ping - Посилає ICMP-пакети для перевірки доступності хоста.
traceroute - Виводить маршрутизацію пакета до хоста.
dig - Отримує детальну DNS-інформацію.
telnet - Встановлює текстове з'єднання з віддаленим хостом.
ssh - Встановлює захищене з'єднання з віддаленим хостом.
scp - Копіює файли між хостами по SSH.
wget - Завантажує файли з вказаної URL-адреси.
nc (netcat) - Універсальний інструмент для роботи з мережевими з'єднаннями.
nmap - Інструмент для сканування мережі та безпеки.
ifconfig - Відображає або налаштовує мережеві інтерфейси.
whois - Запит інформації про домен або IP-адресу.
jq - Обробляє і форматує JSON дані.
top - Відображає інформацію про процеси в реальному часі.
curl - Передає дані з або на сервер за допомогою URL.
